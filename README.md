# Olivia Griffie — Portfolio

This repository contains the source code for my UX/UI design portfolio.  
The site is built with a lightweight custom design system, semantic HTML, and modular CSS.

## Highlights
- Custom design tokens (color, spacing, typography)
- Component-based layout structure
- Case studies written with UX process clarity
- Fully responsive layout
- No external template dependencies

## Tech Stack
- HTML5
- CSS (custom design system)
- Vanilla JavaScript
- GitHub Pages

## Notes
Some initial scaffolding was accelerated using AI tools, but all design decisions, structure, and final implementation were created and reviewed by me.
